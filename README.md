# Netflix Clone

A responsive **Netflix landing page clone** built using **HTML5 and CSS3**.  
This project recreates the user interface of the Netflix homepage to practice modern **layout techniques, responsive design, and UI styling** using pure frontend technologies.

The goal of this project is to improve frontend development skills by replicating a real-world website design without using any frameworks or libraries.

---

# 🚀 Features

- Netflix-inspired landing page design
- Fully responsive layout for different screen sizes
- Hero section with background banner
- Navigation bar with logo and sign-in button
- Movie and TV show sections
- Hover effects on movie thumbnails
- Clean and modern UI

---

| Technology | Purpose |
|-----------|--------|

# 📱 Responsive Design

The layout is designed to work across multiple devices:

- Mobile phones
- Tablets
- Desktop screens

CSS **Flexbox and Grid** are used to create a flexible and responsive layout.
