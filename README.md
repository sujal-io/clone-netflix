# Netflix Clone

A responsive **Netflix Landing Page Clone** built using **HTML5 and CSS3** that replicates the visual design of the Netflix homepage.

This project focuses on recreating a modern streaming platform interface using **pure frontend technologies** without relying on frameworks or libraries. It was developed to practice **layout design, responsive UI development, and real-world website replication**.

The clone demonstrates how modern web interfaces can be built using **semantic HTML, CSS styling, and responsive layout techniques**.

---

## 🚀 Features

- Netflix-inspired landing page interface
- Fully responsive design for multiple screen sizes
- Hero section with a background banner
- Navigation bar with logo and sign-in button
- Movie and TV show preview sections
- Hover effects on movie thumbnails
- Clean and modern user interface

---

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|--------|
| **HTML5** | Provides the structure of the webpage |
| **CSS3** | Handles styling and visual layout |
| **Flexbox** | Aligns and structures UI components |
| **CSS Grid** | Creates responsive layouts |

## 📱 Responsive Design

The layout is designed to adapt across different devices to provide a consistent user experience on:

- Mobile phones
- Tablets
- Desktop screens

Responsive techniques such as **Flexbox, CSS Grid, and media queries** are used to ensure proper alignment and scalability.
